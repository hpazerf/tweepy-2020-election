# tweepy-2020-election

Used the Tweepy API to collect over 17000 tweets about the US 2020 Election over the course of 3 days to look at the lift ratios and sentimental analysis of the presidential candidates
Created this project in a group with 4 people including myself

# Assignment Overview

* Used the Tweepy API to collect 17000 tweets over 3 days about the US 2020 Election.
* Preprocessed the data and found the most common issues discussed over Twitter
* Found the Lift Ratios between the different issues and the presidential candidates
* Found the Sentimental Analysis of presidential candidates with any issues they are heavily related to.
